# Handwriting-Recognition
Немножко о том, что я добавил и как с этим работать.
## Что сейчас есть в проекте?
Сейчас проект устроен следуюзим образом:
 - Всякая гадость, чтоб нормально принимало коммиты без мусора (надеюсь, я правильно все сделал).
 - Папка data с уже сформированной внутри датой разбитой на пакеты.
 - Папка scripts (чуть позже про нее).
 - Папка src.
### Про данные:
Цель первоначальная была создать репу, положить дату, и дать вам доступ, поэтомпу пока что я не стал много качать вариаций. С этим уже можно работать. Сейчас есть готовые датасеты на 60к картинок для тренировки и на 10к картинок для тестирования. В будущем я скорее всего его немножко поменяю/добавлю новые. 

В *_labels лежат "правильные ответы", в *_images лежат сами картинки. 

Файлы уже собраны, так что можно использовать прямо так. У себя проверил, у меня все работает, базовые команды выполняются. Возможно, у вас будут такие же проблемы, как у меня, если у вас нет впна, какие-то пакеты файлов PyTorch скачиваются странновато и он тупит, а вот numpy и весит меньше, и качается без проблем - но все работает с обеими инструментами.

### Про скрипты:
На данный момент в репе лежат прям сами файлы, так как суммарно они весят около 50 мегабайт. Но в гите есть ограничение на размеры файлов + ограничение на общий вес проекта. Возможно, в будущем, когда вариантов датасетов будет много, или когда мы расширимся на буквы (если такое произойдет), коммитить на репу файлы норально будет плохо получаться, скачивание может быть долгим. Чтобы решить эту проблему, я буду писать скрипты для скачивания датасетов на комп с других мест (возможно, сам место сделаю). На крайняк есть вариант сделать гугл док, но если такполучится, то будет лучше.

### Про srt:
Сейчас там лежит файл с загрузсчиком датасета - dataset.py.
Тут просто класс и пара прописанных штук для работы с ним.
