# Softmax Regression для MNIST

Логистическая регрессия с softmax для распознавания рукописных цифр из базы MNIST.

---

## SoftmaxRegression

Реализует мультиклассовую логистическую регрессию (softmax regression) с нуля на NumPy.

**Параметры конструктора:**
| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|----------|
| `learning_rate` | 1.15 | Шаг градиентного спуска |
| `tolerance` | 0.00001 | Критерий остановки: останавливаемся, если изменение весов меньше порога |
| `max_iter` | 1000 | Максимальное количество итераций обучения |

---

### `_softmax(predictions)`

Внутренний метод, применяет функцию softmax к линейным предсказаниям.

| Шаг | Что делает | Зачем |
|-----|-----------|-------|
| `predictions - np.max(predictions, axis=1, keepdims=True)` | Вычитает максимальное значение из каждого предсказания | Численная стабильность: не даёт exp переполниться |
| `np.exp(predictions)` | Возводит e в степень каждого элемента | Превращает скоры в положительные числа |
| `exp / np.sum(exp, axis=1, keepdims=True)` | Нормирует на сумму | Получаются вероятности: каждый образец даёт вектор, сумма элементов = 1 |

**Вход:** массив `[n_samples, n_classes]` с линейными предсказаниями  
**Выход:** массив `[n_samples, n_classes]` с вероятностями для каждого класса

---

### `fit(X, y)`

Обучает модель на данных.

| Переменная | Тип | Описание |
|------------|-----|----------|
| `n_classes` | `int` | Количество уникальных классов (для MNIST = 10) |
| `n_samples` | `int` | Количество объектов в обучающей выборке |
| `n_features` | `int` | Количество признаков (для MNIST = 784 пикселя) |
| `one_hot_y` | `array[n_samples, n_classes]` | One-hot матрица: у каждого образца 1 в столбце правильного класса, 0 в остальных |
| `self.bias` | `array[n_classes]` | Смещение для каждого класса (инициализируется нулями) |
| `self.weights` | `array[n_features, n_classes]` | Веса модели. Строка = признак, столбец = класс (инициализируется нулями) |

**Алгоритм для каждой итерации:**

| Шаг | Формула | Что делает |
|-----|---------|-----------|
| Линейное предсказание | `X @ weights + bias` | Считает скор для каждого класса |
| Softmax | `_softmax(linear_pred)` | Превращает скоры в вероятности |
| Ошибка | `pred_softmax - one_hot_y` | Разница между предсказанием и истиной |
| Градиент по bias | `(1/n_samples) * sum(error, axis=0)` | Усредняет ошибку по образцам для каждого класса |
| Градиент по весам | `(1/n_samples) * X.T @ error` | Матричное произведение признаков и ошибок |
| Обновление bias | `bias -= learning_rate * db` | Шаг градиентного спуска для смещений |
| Обновление весов | `weights -= learning_rate * dw` | Шаг градиентного спуска для весов |
| Критерий остановки | `mean(abs(db - previous_db)) < tolerance` и то же для dw | Если изменение весов меньше порога — обучение завершается досрочно |

---

### `predict(X_test)`

Предсказывает классы для новых данных.

| Шаг | Что делает | Результат |
|-----|-----------|-----------|
| Линейное предсказание | `X_test @ weights + bias` | Скоры для каждого класса |
| Softmax | `_softmax(linear_pred)` | Вероятности для каждого класса |
| Выбор максимума | `np.argmax(softmax, axis=1)` | Индекс класса с наибольшей вероятностью (цифра от 0 до 9) |

**Вход:** массив `[n_samples, n_features]` с тестовыми данными  
**Выход:** массив `[n_samples]` с предсказанными классами (цифрами)
